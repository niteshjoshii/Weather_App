#  Weather Forecast Web Application

A simple weather application that provides real-time weather information for any city using Open-Meteo APIs.

---

##  Features

- Search weather by city
- Real-time temperature display
- Current weather conditions
- Latitude and longitude detection
- Fast and responsive interface
- API-based live weather updates

---

##  Tech Stack

- HTML
- CSS
- JavaScript
- Open-Meteo API

---

## Project Structure

Weather/
│
├── index.html
├── style.css
├── script.js
└── README.md

---

##  How It Works

1. User enters city name
2. Geocoding API fetches city coordinates
3. Weather API fetches current weather
4. Weather details are displayed dynamically

---

## APIs Used

### Geocoding API

Used for latitude and longitude detection.

### Open-Meteo Weather API

Used for fetching current weather data.

Website:

https://open-meteo.com/
